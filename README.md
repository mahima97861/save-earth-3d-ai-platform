# 🌍 Save Earth - 3D + AI Platform

An interactive web application that combines **3D Earth visualization** with an **AI-powered sustainability assistant** to spread awareness about environmental issues and solutions.

---

## 🚀 Features

- 🌎 3D Rotating Earth using Three.js
- 🎬 Interactive UI (Netflix-style cards)
- 💬 AI Sustainability Chat Assistant
- 🔍 Informational Popups (Climate Issues & Solutions)
- 🌿 Clean and modern UI design

---

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- Three.js (3D Graphics)

---

## 🎯 Learning Outcomes

- Basics of 3D rendering using Three.js  
- DOM manipulation and event handling  
- UI/UX design for interactive applications  
- Creating simple AI-like chatbot logic  

---

## 🚀 How to Run

1. Clone the repository  
2. Open `index.html` in browser
